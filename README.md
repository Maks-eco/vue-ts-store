## vue-store

Приложение онлайн магазин, которое содержит три страницы:

- страница со списком категорий и продуктов
- страница продукта
- страница корзины

Данные магазина берутся с помощью [Ecwid REST API](https://api-docs.ecwid.com/reference/rest-api)

## Технологии

- Typescript
- VueJS
- HTML/CSS
